# 📖 Manual do Usuário - PayTrack

Este guia vai te ensinar a usar as principais funções do PayTrack para gerenciar suas vendas de trufas.

---

## 1. Dashboard (Tela Inicial)
Ao abrir o sistema, você vê o **Dashboard**.
-   **Total a Receber:** A soma de tudo que todos os clientes te devem.
-   **Tabela de Saldos:** Uma lista rápida mostrando quem te deve (em vermelho) e quem está em dia (em verde).

## 2. Cadastrando Clientes
Antes de vender, você precisa ter o cliente cadastrado.
1.  Clique em **Clientes** no menu lateral ou no botão **+ Novo Cliente** (se houver atalho).
2.  Preencha o **Nome** (obrigatório) e **Telefone** (opcional).
3.  Clique em **Salvar**.

## 3. Registrando uma Venda
Quando alguém pegar trufas:
1.  Clique no botão **+ Nova Venda** no topo da tela.
2.  **Cliente:** Selecione quem está comprando.
3.  **Quantidade:** Quantas trufas.
4.  **Preço Unitário:** O padrão é R$ 5,00, mas você pode mudar se fizer um desconto.
5.  **Data:** Já vem com a data de hoje, mas pode mudar se estiver lançando algo de ontem.
6.  Clique em **Salvar**.
    *O saldo do cliente será atualizado automaticamente.*

## 4. Registrando um Pagamento (PIX)
Quando cair um PIX na sua conta:
1.  Clique no botão **+ Novo Pagamento**.
2.  **Nome no Comprovante:** Digite o nome que aparece no extrato do banco (ex: "Carlos Eduardo").
3.  **Valor:** O valor exato recebido.
4.  **Data:** A data do pagamento.
5.  **Associar a Cliente:**
    *   **Automático:** Se você deixar nessa opção, o sistema vai tentar achar um cliente com o mesmo nome que você digitou.
    *   **Manual:** Se o nome no banco for muito diferente (ex: "Mãe do João"), selecione o cliente "João" manualmente na lista.
6.  Clique em **Salvar**.

## 5. Conferindo Saldos
Vá na aba **Clientes** ou olhe no **Dashboard**.
-   **Consumido:** Valor total de trufas que a pessoa pegou.
-   **Pago:** Valor total que ela já te transferiu.
-   **Saldo:** A diferença. Se for positivo, ela te deve. Se for zero, está pago.
